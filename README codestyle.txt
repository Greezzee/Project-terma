Весь движок написан в едином кодстайле. Так как игра по сути не является частью движка, вы вольны использовать любой другой.
Однако если вы делаете правки движок, использовать приведённый кодстайл обязательно!

ИМЕНА ПЕРЕМЕННЫХ:
	CONST_DATA - константы
	ClassFunction - функции классов и прочие функции. Имена классов
	_some_private_data - имена private и protected переменных
	some_data - локальные переменные. Также можно использовать для private или protected переменных классов, если они относятся к внешним элементам
	mathIsCool - используется очень редко, в основном в больших математических функциях

СКОБОЧКИ:

Function()
{
	//some code
}

Class()
{
	//some code
}

if() {
	//аналогично for, while, switch, etc
}

Используйте табы, а не пробел!
